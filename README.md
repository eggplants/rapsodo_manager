# Rapsodo Manager

![logo](https://raw.githubusercontent.com/klishai/rapsodo_manager/master/img/logo.png)

## このリポジトリは…

- 野球トラッキングシステム[Rapsodo]の投手データ管理アプリ.
- 筑波大学知識情報・図書館学類システム主専攻3年次授業の作成物です.

## 使用した技術

- 開発
  - GitHub
  - VS Code
  - GNU nano

- バックエンド
  - CGI(Ruby)
  - SQLite3
  - BCrypt

- フロントエンド
  - jQuery
    - [tablePagination]
    - [PACE]
    - [Highcharts]
  - BootStrap
  - CSS
  - HTML5
